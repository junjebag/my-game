# 5주차 실습 기록

## 선택한 템플릿 및 주제
- 템플릿: space_shooter.py
- 주제: 스테이지 클리어 방식의 슈팅 게임 "Galaxy Strike"
- 기존 템플릿의 엔드리스 방식을 스테이지 3개를 순서대로 클리어하면 승리하는 목표 지향형 구조로 변경

## 사용한 AI 프롬프트 (요약)

1. HUD 정렬 문제 해결 요청
   - "오른쪽 위 하트(목숨)가 화면 밖으로 짤려서 안 보여. 하트 3개가 다 보이게 정렬 고쳐줘."

2. 구현 가능성 확인
   - "골드 시스템이랑 상점에서 스킨 구매하는 기능을 pygame으로 구현할 수 있어? 어떻게 만들면 돼?"

3. 패럴렉스 스크롤링 구현 요청
   - "배경 별들이 크기에 따라 속도가 다르게 스크롤되는 패럴렉스 효과를 pygame으로 어떻게 구현해?"

## AI 답변에서 도움이 된 것
- 골드를 json 파일로 저장해서 게임을 껐다 켜도 유지되게 하는 방법
- 음량 슬라이더를 마우스 드래그로 구현하는 방식
- 스테이지별 배경색과 별 색상을 다르게 설정해서 분위기 차별화하는 방법

## AI 답변을 수정하거나 버린 것
- AI가 난이도 선택 시스템 제안 → 내가 스테이지 선택으로 바꿈
- AI가 점수 기반 스킨 해금 제안 → 내가 골드 상점 방식으로 바꿈
